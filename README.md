# AI Dynamic Pathfinding Agent

## Overview
This project implements a grid-based pathfinding agent using:

- A* Search
- Greedy Best-First Search

The application is built using Pygame and includes an interactive graphical interface.

## Features
- Start and Goal selection
- Manual obstacle placement
- Random obstacle generation
- Algorithm toggle (A* / Greedy)
- Real-time metrics:
  - Nodes Expanded
  - Path Cost
  - Execution Time

## Installation

Install dependencies:

pip install pygame

## Run

python main.py

## Controls

- Left Click: Place Start, Goal, Obstacles
- Run Search: Execute selected algorithm
- Reset Grid: Clear grid
- Random Obstacles: Generate random barriers
